# Email-client-app

- A email client app created using ReactJs, Redux-toolkit, CSS3.
- Link to live demo - https://email-app-project.netlify.app/

--- 

### Features
- Email listing.
- Shows email body in master-slave layout.
- Pagination.
- Add to and remove from favorites.
- Filters:
  - Read
  - Unread
  - Favorites
